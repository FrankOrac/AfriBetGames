

2. Home Screen Overview

The home screen displays links to all major game categories:

Main Game

Virtual Game

Noon Game

Night Game


When a user clicks Main, it expands to show:

Minor

Major

Mega


Each link leads to its respective gameplay interface with clear odds, number ranges, and timers.



3. Virtual Betting (Weekly Rounds)

Runs for 35 virtual weeks.

The current week number (e.g., “Week 12 of 35”) appears at the header.

A countdown timer of 4 minutes 59 seconds is displayed at the top for each round’s kickoff time.

Automatically restarts the timer for the next round after each result is displayed.



4. Main Game Modes
The Main section consists of three categories:

Minor

Major

Mega



5. Minor Game Category

Number range: 0 to 40

Odds range: 1.01 to 7.00 (randomized)

Player selection: 5 numbers

Winning condition: 3 or more correct numbers



6. Major Game Category

Number range: 0 to 60

Odds range: 1.01 to 13.00 (randomized)

Player selection: 7 numbers

Winning condition: 3 or more correct numbers



7. Mega Game Category

Number range: 0 to 90

Odds range: 1.01 to 17.00 (randomized)

Player selection: 10 numbers

Winning condition: 5 or more correct numbers



8. Noon Game

Number range: 0 to 25

Odds range: 1.01 to 12.00 (randomized)

Player selection: 3 numbers

Bonus system: 1 bonus number automatically generated by the system

Winning condition: 2 or more correct numbers + the bonus number

Result time: 12:00 PM (daily)



9. Night Game

Number range: 0 to 25

Odds range: 1.01 to 12.00 (randomized)

Player selection: 3 numbers

Bonus system: 1 bonus number automatically generated

Winning condition: 2 or more correct numbers + the bonus number

Result time: 12:00 AM (midnight daily)



10. Outcome and Payout System

Each game round generates random winning numbers.

Payouts depend on:

The number of matches.

The odds assigned to the player’s selection.


Results are displayed in real time after each draw.



11. User Interface and Experience

Header Section: Displays countdown timer and current virtual week.

Navigation Tabs: Main / Virtual / Noon / Night.

In-Game Display: Shows odds, selected numbers, and potential payout before confirmation.

Result Board: Displays winning numbers, bonus (if applicable), and winning payout table.
